Compact Image Captioner via Multi-Teacher Knowledge Distillation
This project develops a lightweight, 158M parameter AI model designed for real-time image captioning on UAVs during disaster response. 
By using a Multi-Teacher Knowledge Distillation framework, the system transfers complex visual reasoning from large models like BLIP-2 and GIT-Large into a compact student model.
Performance: Achieves 146 FPS on edge hardware with a 0.61 BERTScore.Architecture: Combines a DeiT-Tiny vision encoder with a GPT-2 language decoder.
Datasets: Trained on FloodNet and RescueNet to identify flooding and structural damage.
Impact: Enables autonomous, cloud-independent situational awareness in connectivity-limited disaster zones
